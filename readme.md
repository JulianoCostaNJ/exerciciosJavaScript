## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte integrante da disciplina de **Programação Front-End** do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas** O objetivo principal é exercitar e consolidar os conceitos fundamentais da linguagem JavaScript, abrangendo desde a lógica básica até manipulações avançadas de funções e arrays.

O trabalho consiste na implementação de 50 algoritmos que resolvem problemas de lógica de programação, seguindo requisitos técnicos de organização e documentação exigidos pelo Professor José Carlos Domingues Flores.

## ✨ Funcionalidades
A lista de exercícios está dividida em cinco grandes blocos temáticos:
- **Estruturas de Decisão (Condicionais):** Exercícios de 01 a 10 envolvendo verificações de paridade, idade, login, IMC e classificação de triângulos.
- **Estruturas de Repetição (Loops):** Exercícios de 11 a 20 focados em loops `for`, `while` e `do...while`, cálculos de fatorial e sequência de Fibonacci.
- **Funções (Declaração, Parâmetros e Retorno):** Exercícios de 21 a 30 focados em modularização, recursividade e conversões de unidades.
- **Métodos de Arrays e Strings:** Exercícios de 31 a 40 utilizando `.map()`, `.filter()`, `.reduce()`, `.find()` e manipulação de cadeias de caracteres.
- **Funções Modernas (Arrow Functions, Callbacks e HOF):** Exercícios de 41 a 50 explorando Arrow Functions, Closures, Higher-Order Functions e Currying.

## 🛠️ Tecnologias Utilizadas
- **JavaScript (ES6+):** Linguagem principal utilizada para o desenvolvimento de todos os algoritmos.
- **Node.js:** Ambiente de execução para os scripts no console.
- **VS Code:** IDE recomendada para organização e desenvolvimento do projeto.

## 📂 Estrutura do Projeto
Conforme os requisitos técnicos obrigatórios, os arquivos estão organizados individualmente para garantir a clareza e manutenibilidade.

```text
/
├── exercicio01.js  # Verificação de número par ou ímpar
├── exercicio02.js  # Verificação de idade para votação
...
├── exercicio50.js  # Implementação de Currying simples
└── README.md       # Documentação do projeto
```

> **Nota:** Cada arquivo-fonte contém um cabeçalho obrigatório com as informações dos componentes do grupo e uma breve explicação da lógica aplicada para resolver o problema.

## ⚙️ Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/JulianoCostaNJ/exerciciosJavaScript.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd exerciciosJavaScript
   ```

3. **Execute um exercício específico:**
   Certifique-se de ter o (https://nodejs.org/) instalado em sua máquina.
   ```bash
   node exercicio01.js
   ```
